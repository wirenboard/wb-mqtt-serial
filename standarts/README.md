# Стандарты экосистемы Wiren Board

Действующий пакет стандартов на правила формирования имён, текстов и шаблонов в экосистеме Wiren Board.

Все три документа — черновики на 2026-05-25, готовятся к публикации под лицензией CC BY 4.0.

## Состав пакета

| Документ | Тема | Область применения |
| --- | --- | --- |
| [СТП 002-2026](<СТП 002-2026 (MQTT-топики, черновик).md>) | Идентификаторы MQTT-топиков | Любое ПО WB, публикующее в MQTT (драйверы `wb-mqtt-*`, прошивки, `wb-rules`, mobile-app, веб-конфигуратор) |
| [СТП 003-2026](<СТП 003-2026 (UI-тексты, черновик).md>) | Стиль текстов в пользовательском интерфейсе | Любые UI-тексты WB (веб-конфигуратор, mobile-app, MQTT-метаданные, шаблоны, `wb-rules`) |
| [СТП 001-2025 изд.03](<СТП 001-2025 изд.03 (черновик).md>) | Шаблоны устройств для драйвера `wb-mqtt-serial` | Шаблоны `wb-mqtt-serial`; канонические имена сущностей WB |

## Иерархия зависимостей

```
       СТП 002-2026          СТП 003-2026
       (MQTT-имена)          (UI-стиль)
              \                   /
               \                 /
                \               /
                 ▼             ▼
              СТП 001-2025 изд.03
        (шаблоны wb-mqtt-serial,
         канонические реестры имён)
```

СТП 001 ссылается на СТП 002 и СТП 003 как на нормативные источники общих правил. Документы 002 и 003 независимы друг от друга и применимы вне контекста шаблонов `wb-mqtt-serial`.

## Внешние источники

- **[Wiren Board MQTT Conventions](https://github.com/wirenboard/conventions)** — публичные соглашения Wiren Board по MQTT. Раздел 1.2 «Naming Conventions (2024+)» — основа СТП 002; раздел 3 «Control Types» — основа типов контролов в СТП 001.
- **«Текст в веб-интерфейсе»** (`../Текст в веб-интерфейсе.md`) — действующая корпоративная инструкция Wiren Board; первоисточник правил для СТП 002 (часть) и СТП 003.
- **`../wb-mqtt-serial-device-template.schema.json`** — первоисточник структуры шаблона для СТП 001 изд.03.

## Что подавать AI-агенту

| Задача | Требуемый контекст |
| --- | --- |
| Генерация шаблона `wb-mqtt-serial` | СТП 001 + СТП 002 + СТП 003 + JSON-схема шаблона |
| Генерация виртуального устройства `wb-rules` | СТП 002 + СТП 003 |
| Проверка имени MQTT-топика на соответствие конвенции WB | СТП 002 |
| Проверка или перевод UI-текста | СТП 003 |

Этого набора достаточно для решения соответствующей задачи без обращения к внешним ресурсам.
